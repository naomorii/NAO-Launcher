CARDINAL
=====

<img src="https://github.com/user-attachments/assets/1fca4d38-b4de-4482-a3c7-b3eb7f5061cb" width="100" />

This app is a fan-based Android app Launcher, based on Sword Art Online.

When the gesture (defauld : swipe down) is used, the launcher appears in the foreground for you to start the app you want, without having to look for it.
As soon as the chosen app starts, the launcher collapses and closes itself.


Compatibility
-----

This app has been written to work on Android from version 7.0 (Nougat).


Installation
-----

1 - Download the files.

2 - Extract the files from the archive.

3 - Open the app-release.apk file.
    You may be asked to grant access to your file manager for the installation of .apk files from unknown sources.
    Do it as it is asked and the launcher will be installed.
